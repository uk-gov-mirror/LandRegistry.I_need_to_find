Find property information

Our vision is: "Providing the right data, at the time right time, in the right way".

By improving or replacing our current land or property information services- we're testing the assumption that we'll help:

  • improve customer experience
  • reduce number of calls to our citzen support centre
  • reduce technical debt

The code for this prototype is in this repository. (https://github.com/LandRegistry/I_need_to_find)

Version history

Version 5 07 May 2019

    Last tested version

    Use postcode: PL1 3SB (8 Peel Street) or EX4 1AX (3 Clinton street).

    Scenarios we tested

      We've added:
    • journey to buy copies of the register, title plan and the deeds
    • options to download documents and view online versions
    • a guidance page to give more information about the documents available to buy
      Based on round 5 user research we've:
    • iterated on the initial start page design
    • provided Plain English guidance

Version 4 26 April 2019

    Account creation and sign in

    Use postcodes: PL1 3SB (8 Peel Street), EX4 1AX (3 Clinton street) or BS6 6RU (10 Elgin Park).

    Scenarios we tested

      We've added:
    • account creation and sign in pages
      Based on round 4 user research we've:
    • separated easements and restrictive covenants back out into separate entries and provided legal definitions of these

Version 3 12 April 2019

    Introducing more property information

    Use postcodes: PL1 3SB (8 Peel Street), EX4 1AX (3 Clinton street) or BS6 6RU (10 Elgin Park).

    Scenarios we tested

      We've added:
    • real addresses
    • Price paid data with links to the HMLR service
    • House Price Index data with links to the current service
    • planning permission information with the intention to link to Search for local land charges
    • option to download relevant deed when mentioned as 'copy filed' in 'other entries'
      Based on round 3 user research we've iterated the:
    • layout to a single column for the property page
    • language used throughout to simplify it
    • easements and restrictive covenants section to read 'other entries'
    Alternate version (12 April 2019)

    Same as Version 3 but with different Start page content to show summaries of the pay tiers instead of bulleted lists.
