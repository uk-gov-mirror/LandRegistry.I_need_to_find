Search for land and property information Providing the right land or property data, at the right time, in the right way

By improving or replacing our current land or property information services- we're testing the assumption that we'll help:

  • improve customer experience
  • reduce number of calls to our citzen support centre
  • reduce technical debt

The code for this prototype is in this repository. (https://github.com/LandRegistry/I_need_to_find)

Version history

Beta

MVP v1
Search by postcode or street name and view a free summary of information about an address

Scenarios we're testing include:

  • can users find the results they need even if the addresses appear to be the same? Search by street name- use Peel Street, choose Peel Street Runcorn, and then 2 Peel Street, Runcorn

  • can users find the result they need in a long list of results? Search by postcode- use EX4 1AX, choose 35 Clinton Street, Exeter, EX4 1AX

  • what are users expecting to see if we don't have any information for a property they're looking for? Search by street name- use Peel Street, choose Peel Street Runcorn, and then 3 Peel Street, Runcorn

  • can users navigate between pages of results? Search by street name or postcode- use Peel Street, choose Peel Street Runcorn, and then 1,2 or 3 Peel Street Runcorn or EX4 1AX and 35 Clinton Street

View user insights (link opens in new tab) from testing this prototype.

Paused: Search, pay and view end to end journey (not tested) August-2019

Use postcode: PL1 (Peel Street), EX4 (Clinton street) or BS6 (Elgin Park)

In this prototype the user can:

  • search for a property with a postcode or street name
  • view a free summary of information about an address
  • create an account and pay for access to more information
  • view a summary of the property title including the title number
