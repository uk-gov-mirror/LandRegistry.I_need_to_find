Introduction

I need to find out about land or property's vision is: Providing the right data, at the time right time, in the right way

The code for this prototype is in this repo. (https://github.com/LandRegistry/I_need_to_find)

Current prototype

Version 5

  • 07-05-2019

    Scenarios we tested

    Use postcode: PL1 3SB (8 Peel Street) or EX4 1AX (3 Clinton street)

      We've added:
    • journey to buy copies of the register, title plan and the deeds
    • options to download these documents and view online versions
    • a guidance page to give further information on the documents available to buy
      Based on round 5 user research we've:
    • iterated on the initial start page design
    • provided Plain English guidance

Version 4

  • 26-04-2019

    Scenarios we tested

    Use postcodes: PL1 3SB (8 Peel Street), EX4 1AX (3 Clinton street) or BS6 6RU (10 Elgin Park)

      We've added:
    • account creation and log in
      Based on round 4 user research we've:
    • separated easements and restrictive covenants back out into separate entries and provided legal definitions of these

Version 3

  • 12-04-2019

    Scenarios we tested

    Use postcodes: PL1 3SB (8 Peel Street), EX4 1AX (3 Clinton street) or BS6 6RU (10 Elgin Park)

      We've added:
    • Price paid data with links to the HMLR service
    • House Price Index data with links to the current service
    • planning permission information with the intention to link to Search for local land charges
    • option to download relevant deed when mentioned as 'copy filed' in 'other entries'
      Based on round 3 user research we've changed the:
    • layout to a single column for the property page
    • language used throughout to simplify it
    • easements and restrictive covenants section combined to read 'other entries'
  • 12-04-2019 (alternate version)

    Same as Version 3 but with different Start page content to show summaries of the pay tiers instead of bulleted lists.

Version 2

  • 08-04-2019

    Scenarios we tested

    Use any property number for postcodes: PL1 3SB, EX4 1AX or BS6 6RU.

    • Content updated based on week 2 user testing feedback
    • Outline of changes made to each page- why we've changed them, what we've kept the same and what we're testing
    • Options for introducing easement and restrictive covenant indicators

Version 1

  • 29-03-2019
