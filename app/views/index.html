I need to find out about land or property

This is the prototype homepage for "I need to find out about land or property" project at HM Land Registry.

The code for this prototype is in this repo. (https://github.com/LandRegistry/I_need_to_find)

Version history

Version 4 (WIP)

  • 26-04-2019
      We've added:
    • account creation and log in
    • options to purchase copy of the register, title plan and copies of any deeds referred to in the register
      Based on round 4 user research we've:
    • iterated on the initial start page design
    • gone back to having easements and restrictive covenants as separate entries

    Version 3

    • 12-04-2019
        We've added:
      • Price paid data with links to the HMLR service
      • House Price Index data with links to the current service
      • planning permission information with the intention to link to Search for local land charges
      • option to download relevant deed when mentioned as 'copy filed' in 'other entries'
        Based on round 3 user research we've changed the:
      • layout to a single column for the property page
      • language used throughout to simplify it
      • easements and restrictive covenants section combined to read 'other entries'
    • 12-04-2019 (alternate version)

      Same as Version 3 but with Start page content to show summaries of the pay tiers instead of bulleted lists
